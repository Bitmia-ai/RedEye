---
description: "Show recent iteration history"
argument-hint: "[count]"
---

Read .redeye/changelog.md from the project root.

If an argument was provided, use it as the number of entries to show. Default: 5.

Show the last N entries from .redeye/changelog.md, formatted clearly.

If the file is empty or has no entries, say: "No iterations completed yet. Run /redeye:start to begin."
